⚙️ Architecture & Deployment

This SPA uses client-side routing via the History API + popstate events, a modular data layer (products.js), component-style render functions, localStorage cart persistence, and lazy-loaded product images. Deploy to Vercel (drop the folder), Netlify (drag-and-drop the zip), or Render (static site, render.yaml included). The vercel.json and netlify.toml configs are bundled in this zip.
