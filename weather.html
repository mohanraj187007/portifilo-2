⚙️ How this works

This dashboard demonstrates Asynchronous JavaScript & RESTful API concepts: using fetch() with async/await to call the OpenWeatherMap /weather and /forecast endpoints, comprehensive try/catch error handling for failed network requests, dynamic parsing of nested JSON (coordinates, weather arrays, sys object), and real-time DOM rendering — all without any framework.
