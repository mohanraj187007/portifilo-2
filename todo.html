    Under the Hood

    How It Works

    💾

    localStorage Persistence

    Every change is saved immediately to window.localStorage. Reload the page — your tasks are still there.

    🔁

    Full CRUD

    Create tasks with priority, read and filter them, update text inline via the ✏️ button, delete individually or clear all completed at once.

    🎛️

    State-Driven Render

    A single render() function reads the tasks array and current filter, rebuilds the DOM. Predictable, debuggable state management.

    🖱️

    Delegated Events

    Edit, toggle, and delete all use delegated listeners — efficient, no memory leaks from repeatedly re-binding on re-render.

    ↕️

    Drag to Reorder

    Grab any task and drag it into a new position. Order is saved back to localStorage instantly.

    ⚡

    Priority System

    Three priority tiers — High (red), Medium (yellow), Low (cyan) — with colour-coded accent bars and badges on every task.
